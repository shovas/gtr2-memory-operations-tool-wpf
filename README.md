## I have not stopped developing this tool

- See the dev branch for active development
- I'm using the dev branch because everything is so messy right now but a lot
  is working so far including a log tab, a shared memory tab (ListView and
  searchable), a program memory tab (ListView, searchable, and memory can be
  edited and written back to live GTR2 memory), and a settings tab.
- I am getting to the point where the start of my real goal is almost in sight:
  Automatic AI - An algorithm that watches Player and AI performance and
  adjusts AI performance levels to the Player. It won't be just arcade-style
  rubberbanding AI, although eventually that will be one of the selectable
  strategies. Rather, I intend to have a suite of a few smart algorithms that
  won't feel like rubberbanding.
